About TeaOS

TeaOS is a modern Linux operating system built on Fedora using bootable container technology. It focuses on simplicity, performance, and a clean developer experience.

Features

Immutable System

Reliable system updates using container-based deployments.

Fedora Based

Built on the latest Fedora technology and ecosystem.

GNOME Desktop

Modern desktop experience with a clean interface.

Developer Friendly

Perfect environment for hacking and building software.
